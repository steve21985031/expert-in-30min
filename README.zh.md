# expert-in-30min

> 用大约 30 分钟，把陌生领域压缩成一个像专家一样能讲、能写、能展示的输出框架。

[English](./README.md)

## 它是什么

`expert-in-30min` 是一个偏工作流型的 skill，用来把陌生主题快速压缩成“结构化判断 + 专家感表达 + 双文件交付”。

它适合这些场景：

- 准备分享、直播、播客、培训
- 写课程大纲、顾问 briefing、销售叙事
- 把零散资料压缩成强观点
- 生成一个统一结构化文档和一个对应的 HTML 展示页

## 它会产出什么

默认产出两份严格对应的结果：

1. `report.md`：唯一主文档
2. `index.html`：同结构的漂亮展示页

两者章节一一对应。HTML 不是第二套内容系统，而是主文档的可视化层。

## 启动方式

这个 skill 默认先问两个问题：

1. 你想快速进入的学习领域是什么？
2. 你希望输出采用哪种风格？
   - `学院派`
   - `咨询派`
   - `IP派`

## 默认输出章节

- 基础信息
- 30 秒外行解释版
- 3 分钟专业解释版
- 领域骨架
- 术语表
- 主流观点
- 核心争议
- 过时说法
- 表达模板
- 高频问答
- 风险边界
- 行动建议

## 为什么值得单独做成一个 skill

大多数人不是找不到资料，而是找到了也讲不出来、写不出来、卖不出去。

这个 skill 解决的不是“搜索”本身，而是把信息压缩成一个能快速建立确定感、结构感和表达力的框架。

## 文件说明

- `SKILL.md`：可执行入口与运行规则
- `Spec-expert-in-30min.md`：产品与工作流设计稿
- `references/`：输出模式、包装规则、搜索策略、风险边界
- `scripts/`：辅助脚本，比如自动打开 HTML
- `assets/images/`：公开仓库里直接展示的图示和信息图

## 备注

- 这个 skill 服务的是“快速建模与表达”，不是替代受监管行业的正式专业意见。
- 医疗、法律、金融等高风险领域，仍然需要额外核验。
