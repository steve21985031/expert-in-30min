# Output Package

expert-in-30min 的默认交付物应是一个统一双文件结果，而不是单段聊天文本，也不是多个零散素材文件。

## 推荐目录

```text
outputs/<timestamp>-<topic-slug>/
├── report.md
└── index.html
```

## 文件说明

### report.md

唯一的结构化主文档。

要求：

- 它是所有内容的唯一事实源
- 所有章节必须稳定
- HTML 必须按它的结构进行映射

默认章节顺序：

1. 基础信息
2. 30 秒外行解释版
3. 3 分钟专业解释版
4. 领域骨架
5. 术语表
6. 主流观点
7. 核心争议
8. 过时说法
9. 表达模板
10. 高频问答
11. 风险边界
12. 行动建议

### index.html

主入口页面。默认在浏览器打开。

建议模块：

- 头部标题与风格标签
- 30 秒解释版
- 3 分钟解释版
- 领域骨架
- 术语表
- 主流观点 / 核心争议 / 过时说法
- 表达模板句库
- FAQ 折叠卡片
- 风险边界提醒
- 行动建议区

## 生成原则

- `report.md` 面向编辑和归档
- `index.html` 面向浏览和展示
- 二者必须使用同一套章节结构和同一套内容顺序

避免生成多套彼此不一致的内容。默认只维护一份主文档和一份展示页。
